Utente Web Interface2

Verdaccio offre un'interfaccia web utente per mostrare solo i pacchetti privati e può essere personalizzata,.

web:
  enable: true
  title: Verdaccio
  logo: logo.png
  scope:

Tutte le restrizioni di accesso definite per proteggere i pacchetti si applicano anche all'interfaccia Web.

Configurazione

ProprietàTipoRichiestoEsempioSupportoDescrizione
enablevariabile booleanaNovero/falsotuttiabilita l'interfaccia web
titlestringaNoVerdacciotuttiDescrizione del titolo HTML
logostringaNohttp://my.logo.domain/logo.pngtuttiun URI in cui si trova il logo
scopestringaNo\@myscopetuttiSe si utilizza questo registro per uno specifico scope, definisci quello scope nelle istruzioni dell' intestazione dell'interfaccia web utente (nota: escape @ with \@)
