title: 'Developing Plugins'
---

There are many ways to extend `verdaccio`, the kind of plugins supported are:

- [Authentication](plugin-auth.md)
- [Middleware](plugin-middleware.md)
- [Storage](plugin-storage.md)
- [Theme](plugin-theme.md)
- [Filter](plugin-filter.md)

:::info
We recommend developing new plugins using our [Plugin Generator](plugin-generator.md) as a starting point.

For more details, check the individual plugin pages and [typescript type definitions](https://github.com/verdaccio/verdaccio/tree/master/packages/core/types). You can also learn a lot from other plugin solutions which you find via the [plugin search](https://www.verdaccio.org/dev/plugins-search).
:::
